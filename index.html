How to Set Up Thunderbird with demo@chhunmeyling.com

Step 1: Download Mozilla Thunderbird

Download Thunderbird

Run the installer after downloading. Follow the on-screen instructions to complete installation.

Launch Thunderbird. On the welcome screen, choose 'Set up an existing email account'.

Enter the following details:
Your Name: Demo User
Email Address: demo@chhunmeyling.com
Password: [Your Password]

Choose Manual Config and use the following settings:

  • Incoming: IMAP, mail.chhunmeyling.com, Port: 993, SSL: SSL/TLS
  • Outgoing: SMTP, mail.chhunmeyling.com, Port: 465, SSL: SSL/TLS
  • Username: demo@chhunmeyling.com (for both)
